**Xem báo cáo:** mở `index.html` hoặc truy cập GitHub Pages của repo.

## Nội dung
- **Tổng quan:** tỉ lệ giao trễ, tỉ lệ hủy, GMV đơn trễ, đơn treo.
- **Vận chuyển:** tỉ lệ trễ theo từng đơn vị vận chuyển.
- **Top 10 cửa hàng** nhiều đơn trễ nhất (chi tiết theo ĐVVC + phân bố ngày trễ).
- **Đơn treo xử lý:** trạng thái, tuổi treo, nguyên nhân, cửa hàng.
- **Khuyến nghị hành động.**
